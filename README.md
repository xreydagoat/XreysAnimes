# Xreys Animes

A black anime-library website for Xreys Animes.

## Files

- `index.html` - page structure
- `styles.css` - visual design
- `app.js` - show data, search, modal, and episode list behavior

## Deploy On Netlify

1. Go to https://app.netlify.com/drop
2. Drag this repository folder or connect the GitHub repo in Netlify.
3. Netlify will publish the static site.

## Note

This public repository does not include local episode video files. The public build is a library preview.
